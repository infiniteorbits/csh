

Abbreviations
----------------------------------

The following abbreviations are used in command names:


.. class:: table

.. list-table::
   :widths: 32 68 
   :header-rows: 1

   * - Abbreviation
     - Description

   * - cnt
     - count

   * - ch
     - channel

   * - curlim
     - current limit

   * - uv
     - undervoltage

   * - ov
     - overvoltage

   * - oc
     - overcurrent

   * - hyst
     - hysteresis

   * - temp
     - temperature

   * - wdt
     - watchdog
